## Run it

Use the following steps to run this Project locally:

- Create a directory

- Open the terminal

- Run this command:
`git clone git@github.com:Draxeytina/hello-react-rails.git`

- Enter in to the folder:
`cd hello-react-rails`

- Open the folder with a code editor
